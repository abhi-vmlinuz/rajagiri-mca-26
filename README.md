# MCA Curriculum Notes

A personal repository tracking my 2-year MCA (Master of Computer Applications) journey at Rajagiri. This is where I keep assignments, code, and notes for every course across the program.

## What's Inside

The repo is organized by course. Each top-level directory corresponds to a subject or course from the curriculum.

```
Rajagiri/
  BrideCourse/          - bridge course material (pre-MCA foundation)
    C-programming/
      SIMPLE/           - basic I/O, arithmetic, conversions
      IF-ELSE/          - conditionals and switch-case
      WHILE/            - loop-based programs
  <future-courses>/     - more courses will be added as the program progresses
```

## Compiling C Programs

Any `.c` file can be compiled with gcc:

```
gcc -o output_name source.c
./output_name
```
